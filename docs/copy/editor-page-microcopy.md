# /editor Microcopy Pack (EN)

## Upload Zone
- Default: **Drop your image here, or click to upload**
- Helper: JPG, PNG, WEBP · Max 10MB
- Secondary: No signup required to start

## Mode Selector
- Auto Enhance
- Background Remove
- Smart Rewrite

## Action Buttons
- Primary: **Generate**
- Secondary: **Try Another Version**
- Tertiary: **Reset**

## Loading States
- Uploading: Uploading your file...
- Processing: Creating your result...
- Queue: High traffic right now. Your result is in queue.

## Success States
- Done: Your edit is ready.
- Export ready: Download is ready.
- Rewrite ready: 3 copy options are ready.

## Empty States
- No file: Upload an image to start editing.
- No result yet: Pick a mode and click Generate.
- History empty: No recent edits yet.

## Error Messages
- File type: Unsupported file type. Please upload JPG, PNG, or WEBP.
- File size: File too large. Max upload size is 10MB.
- Network: Connection issue. Please try again.
- Timeout: Processing took too long. Try a smaller image or retry.
- Abuse limit: You’ve reached your current limit. Please try later or upgrade.
- Unknown: Something went wrong. Please retry.

## Upgrade Nudges
- Free limit hit: You’ve reached today’s free limit.
- Benefit line: Upgrade for more edits, HD exports, and no watermark.
- CTA: Upgrade to Pro

## Confirmation / Safety
- Delete history: Remove this item from your history?
- Replace file: Uploading a new file will clear current results. Continue?

## Accessibility Labels
- Upload button aria-label: Upload image file
- Generate button aria-label: Generate edited result
- Download button aria-label: Download current result
